# **Student Course Registration System Using PHP**

**Student Course Registration System Using PHP** is a web-based application designed for educational institutions to manage student registrations for various courses. The system allows students to register for courses, view available courses, and check their enrollment status. Built using **PHP** for the back-end logic, **MySQL** for database management, and basic **HTML/CSS** for the front-end, this app provides a streamlined and user-friendly system for managing course registrations.

## **Installation**

To set up the **Student Course Registration System Using PHP** locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Momen9Sarsour/Student-Course-Registration-System-PHP.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd Student-Course-Registration-System-PHP
   ```

3. **Set up the Database**:
   - Create a database named `course_registration` in MySQL.
   - Import the database schema from the provided `database.sql` file to set up the necessary tables.

4. **Set up your server**:
   - Install **XAMPP** or **WAMP** (if not already installed).
   - Place the project folder in the `htdocs` (for XAMPP) or `www` (for WAMP) directory.

5. **Update the database connection**:
   - Edit the `config.php` file to set your MySQL credentials (username, password, etc.).

6. **Run the app**:
   - Open your browser and go to `http://localhost/Student-Course-Registration-System-PHP/` to access the app.
